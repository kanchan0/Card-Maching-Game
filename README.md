# Card-Maching-Game

## Card matching game is an Objective C project in which you pay a penality everytime you withdraw a card and if consequent matches or not points are awarded as follows
         1. MISMATCH_PENALITY = 2;
         2. MATCH_BONUS = 4;
         3. COST_TO_CHOOSE = 1
